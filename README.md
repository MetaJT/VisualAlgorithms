# VisualAlgorithms
Visual representation of different sorting algorithms [Only Insertion Sort for now]

Buttons:
- [P] = Pause
- [Space] = Stop sort
- [R] = Reset sort
- [UP] = Speed up sort
- [DOWN] = Slow do sort
- [RIGHT] = Increase items to sort
- [LEFT] = Decrease items to sort
- [B] = Bubble Sort
